# Purpose

Creates a simple servlet that includes JQuery, bootstrap, DataTables, jsPlumb, and aceeditor.

These are all included in the bundle that is created, and can be accessed via the following in your head tag:

* jQuery 3.3.1
* jQuery-UI 1.12.1
* jsPlumb 2.8.6
* Ace Editor 1.4.2
* moment 2.23.0
* BootStrap 4.1.3 
* ckeditor 5 11.2.0
* DataTables 1.10.18
* FullCalendar 3.9.0
* JavaScript Tmpl
